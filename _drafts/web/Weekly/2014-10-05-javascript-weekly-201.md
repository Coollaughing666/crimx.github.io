生病修(shui)养(jiao)了几个星期，又是一条好汉！

1. 现在就该开始用的 5 个 Array 方法  
   <http://colintoh.com/blog/5-array-methods-that-you-should-use-today>  
   介绍了 5 个 ECMAScript 5 引入的好用方法，indexOf, filter, forEach, map 和 reduce，主流浏览器都支持的（除了 IE 8 及以下）

1. AngularJS 教程：10,000 字详细指南  
   <http://www.airpair.com/angularjs>  
   Google 开发专家 Todd Motto 创作的史诗级文章，介绍了 Angular 主要的概念，从模块和作用域到指令和过滤器。

1. 用 JavaScript 开发苹果 OS X 应用  
   <http://tylergaw.com/articles/building-osx-apps-with-js>  
   OS X Yosmite 上新的“JavaScript 自动化”功能为 JavaScript 提供了访问原生 OS X 框架的权限。Tyler Gaw 演示了怎么利用它来开发完整的 OS X 应用。

1. 爱屋不及乌 - 理性看待 JS 框架  
   <http://netpoetica.com/why-i-dont-want-your-javascript-framework-but-i-love-you/>  
   这是一篇在 Twitter 和 Reddit 上火起来的文章，反映了选择框架前先对应用做好规划的价值。技多不压身，多收集各种工具不要绑死在一个框架上。

1. Node 和 JS 开发者必备的 3 个 Sublime Text 插件  
   <http://scottksmith.com/blog/2014/09/29/3-essential-sublime-text-plugins-for-node-and-javascript-developers/>  
   也就是耳熟能详的 JSFormat, JSHint 和 JavaScriptNext 。

1. Ember 开发者的一天 - 10 月 6 日 Yehuda Katz 在线约定你  
   <http://nomadjavascript.com/>  
   Ember 项目的 Yehuda Katz 将直播一场在线谈话，介绍一个 Ember 开发者在现实中开发应用时怎么度过一天。
